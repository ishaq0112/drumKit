# Drum Kit Web App
An interactive drum kit web application built using HTML, CSS, and JavaScript. 
Users can play different drum sounds using keyboard keys or by clicking on the buttons.

## Features
- Play sounds using keyboard (W, A, S, D, J, K, L)
- Clickable drum buttons
- Smooth UI design with hover effects
- Responsive layout

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run
1. Clone the repository
2. Open index.html in your browser
